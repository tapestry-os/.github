# Programming Physical Reality.

Tapestry OS is an open-source, scale-invariant operating system 
for physically reconfigurable matter: drone swarms, warehouse 
robot fleets, surgical microrobots, and on a longer horizon, 
molecular machines.

Until now, every one of these systems rebuilds the same coordination 
software from scratch. Tapestry abstracts the applications from the 
underlying physics hardware and figures out how to enable and execute 
applications across a collective of these elements. 

## What is here

**[tapestry](https://github.com/tapestry-os/tapestry)** — 
The core OS. L1–L5 fully implemented on Zephyr. L6–L7 
specified with functional stubs. Apache 2.0.
